# Código Secreto - Simulador de Lógica Deductiva

Este proyecto es una implementación digital del clásico juego de mesa de ingenio, diseñada específicamente como recurso didáctico para el ámbito educativo técnico.

## 🎯 Objetivo del Juego
El desafío consiste en descifrar un código oculto de **4 colores** generado aleatoriamente por el sistema. El jugador dispone de un máximo de **10 intentos** para lograrlo.

## 📜 Reglas y Funcionamiento
Por cada intento realizado, el simulador proporciona pistas visuales basadas en la precisión de la combinación propuesta:

1. **Ficha Roja (Acierto Total):** Indica que un color es correcto y se encuentra en la posición exacta.
2. **Ficha Blanca (Acierto Parcial):** Indica que el color forma parte del código, pero está en una posición incorrecta.
3. **Sin Ficha:** Indica que el color no pertenece a la combinación secreta.

> **Importante:** El orden de las fichas de pista es aleatorio y no se corresponde posición a posición con las fichas del intento. La deducción de qué pista pertenece a cada color es parte del desafío lógico.

## 🏫 Aplicación Pedagógica (UTU)
Este recurso está orientado a fortalecer las siguientes competencias en estudiantes de informática y tecnología:

* **Lógica Proposicional:** Relación entre premisas y conclusiones mediante el uso de conectores lógicos (AND, OR, NOT).
* **Pensamiento Computacional:** Aplicación de algoritmos de búsqueda y descarte sistemático de errores.
* **Resolución de Problemas:** Gestión de la frustración y análisis de datos para la toma de decisiones.

## 💻 Instalación y Uso
Para visualizar el proyecto localmente o subirlo a una plataforma de hosting (como GitHub Pages):

1. Descargar los archivos `index.html`, `style.css` y `script.js`.
2. Mantener los tres archivos en la misma carpeta raíz.
3. Abrir el archivo `index.html` en cualquier navegador web moderno.
